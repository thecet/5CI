#### Descrizione

___

Il seguente script si occupa di eseguire uno scan degli hosts dati per indirizzo ip per controllare le porte aperte.

#### Prerequisiti

___

* Nmap installato e aggiunto al path.
* libreria python-nmap installata.
* python3 installato.
